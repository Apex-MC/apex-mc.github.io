Discord Bot:

We made this discord bot because some people, including us need help to get more players to join our discord servers. With our bot, we let players across the world get more users on their server by advertising as a team to get more players. We also have a set of commands like a!say, a!ddos and more fun commands that we are sure that lots of people would like! The server is currently not 24/7 but soon we will do it 24/7 once there is much more players that want the discord bot to be available at all times. Some stuff are automatic but we still need some developing help! We are working on it everyday and hope to make one of the best bots in the history of Discord.
